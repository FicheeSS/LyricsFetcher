# LyricsFetcher
This scrpit is made to find any lyrics with any supported music file.

# How to

Put the full script into a folder with music in it (the music can be in a subfolder to).

Run : 
```
python main.py
```
or 
```
python3 main.py
```
 # Dependecies 
  -[Mutagen](https://github.com/quodlibet/mutagen)
